# Glowbyte case
 
## Для запуска приложения:

```
cd client
streamlit run app.py
```

WEB-приложение открывается на localhost
На вход ожидается файл с данными, которые необходимо предсказать
После добавления файла можно посмотреть информацию в нем и увидеть график предсказаний

Если существуют истинные значения выходного продукта, то можно проверить точность предсказаний
Загружаем второй файлик для проверки и видим, как один график отличается от другого, а также метрику MAE
